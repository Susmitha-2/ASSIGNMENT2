Centered Container

This container should be centered on the page.

ISSUE: issue in the provided CSS code is that the container has a percentage-based width (width: 50%;) without a specified height and the margin: auto; SOLUTION:there is no body tag in the css code so I added a body style tag with display: flex;, align-items: center;, and justify-content: center; to center the container both horizontally and vertically. I set the height: 100vh; on the body to make sure the container is centered even if the content is not enough to fill the entire viewport. I also added text-align: center; to the .container style to center the content within the container.
